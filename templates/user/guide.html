7. OpenClaw 配置

在 OpenClaw 配置文件 ~/.openclaw/openclaw.json 中添加以下内容：

{
  "models": {
    "providers": {
      "token-relay": {
        "baseUrl": "{{ api_base_url }}",
        "api": "openai-completions",
        "apiKey": "YOUR_API_KEY",
        "models": [
          {% for m in models %}{
            "id": "{{ m.name }}",
            "name": "{{ m.name }}",
            "api": "openai-completions"
          }{% if not loop.last %},{% endif %}
          {% endfor %}
        ]
      }
    }
  },
  "agents": {
    "defaults": {
      "models": {
        {% for m in models %}"token-relay/{{ m.name }}": {}{% if not loop.last %},
        {% endif %}{% endfor %}
      },
      "model": {
        "primary": "token-relay/{{ models[0].name if models else 'model-name' }}"
      }
    }
  }
}

注意：在 agents.defaults.models 中使用 token-relay/模型名 格式，在 agents.defaults.model.primary 中设置默认使用的模型。
